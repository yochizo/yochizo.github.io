Octopress記事の投稿手順

octopressの記事投稿手順を簡単にまとめておく

あたらしく投稿する

octopress/source/_post ディレクトリ配下に “YYYY-MM-DD-<title>.markdown” というファイルができるので、これを適当なテキストエディタで編集する。

プレビューで記事を確認

ブラウザで http://localhost:4000/ にアクセスし、記事の内容を確認します。

記事を公開する

Github Pagesに記事をデプロイします。
